# ssp-chinese2english

A curated Chinese-to-English daily translation practice dataset in Markdown format.

## Overview

This repository contains structured C2E (Chinese-to-English) daily practice materials across three seasons, converted from source PDFs into clean Markdown for easier reading, search, and downstream use.

## Repository Structure

- `pdf/` - Original public-source PDF materials (S1, S2, S3)
- `result/` - Final Markdown outputs
  - `C2E-S1.md`
  - `C2E-S2.md`
  - `C2E-S3.md`
  - `s1_days/`, `s2_days/`, `s3_days/` (day-split files)
- `case/` - Reference examples used during formatting/alignment
- `tools/` - Conversion and verification scripts

## Notes

- Source PDFs are publicly available materials collected for learning and practice.
- Markdown files have been manually reviewed and iteratively corrected for translation quality.
