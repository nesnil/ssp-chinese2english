# ssp-chinese2english

本项目收录上海学生英文报 SSP（Shanghai Students' Post）发布的《中译英 每日一练》资料，包含 PDF 原始版本与人工校对后的 Markdown 版本，便于检索、学习与二次整理。

## 项目简介

- 数据内容：中译英每日练习题与参考答案
- 数据形态：原始 PDF + 结构化 Markdown
- 适用场景：英语学习、教学素材整理、文本检索与轻量数据处理

## 当前进度

项目持续更新中，目前已更新至第三季（S3）。

## 目录结构

- `pdf/`：原始 PDF 资料（S1 / S2 / S3）
- `result/`：整理后的 Markdown 成果
  - `C2E-S1.md`
  - `C2E-S2.md`
  - `C2E-S3.md`
  - `s1_days/`、`s2_days/`、`s3_days/`（按天拆分版本）
- `case/`：格式参考与对照样例
- `tools/`：转换、校验与合并脚本
